### Welcome

Welcome to the **Python101** repository!  
This repository is designed to support the Python101 course taught at the university level. It serves as a central place for lecture materials, examples, and small automation tools used in this course. Whether you are reviewing concepts, following along with lectures, or exploring extra examples, this repo is here to help you build a strong foundation in Python.

### About the Course

**Python101** is an introductory programming course focused on teaching the fundamentals of Python.  

The goal of the course is not only to teach Python syntax, but also to develop problem-solving skills, programming mindset and language independent programming skill that can be applied to later courses and projects.
